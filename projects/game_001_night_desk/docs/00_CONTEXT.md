# Night Desk 项目上下文

## 项目定位

- 项目编号：`game_001_night_desk`
- 中文名：`旅馆前台：请勿让他们入住`
- 英文名：`Night Desk`
- 平台：itch.io HTML5
- 类型：规则判断、短篇恐怖、轻叙事
- 当前阶段：项目初始化与最小可玩骨架

## 当前核心假设

玩家是午夜旅馆前台值班员。每位旅客都会提供入住信息，但值班规则混合了普通登记要求和异常生存规则。玩家需要观察有限信息，选择允许入住或拒绝入住。

本阶段先验证最小循环：

`阅读规则 -> 查看旅客资料 -> 允许或拒绝 -> 获得反馈 -> 进入下一位旅客 -> 触发结局`

## 已确定边界

- 使用原生 HTML、CSS、JavaScript，不引入外部依赖。
- 规则、旅客、结局放在独立数据文件中。
- 当前只有三位测试旅客，用于验证代码流程，不视为正式内容量。
- 暂不加入存档、复杂动画、随机生成、音频系统或多日流程。
- 项目源码只放在 `projects/game_001_night_desk/`。
- 未来上传包只放在仓库根目录 `releases/`。

## 待后续确认

- 正式版本体验时长目标：建议 10-20 分钟。
- 正式旅客数量：建议首版 10-14 位。
- 是否需要多晚值班结构：建议 MVP 暂不做。
- 美术方向：建议低成本复古监控、纸张 UI、轻微 CRT 质感。
- 音效方向：建议先补环境底噪、按钮、门铃、异常提示四类。
