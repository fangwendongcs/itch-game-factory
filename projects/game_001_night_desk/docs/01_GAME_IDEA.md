# 01 Game Idea

## 一句话电梯演讲

你是旅馆夜班前台：预订信息可以伪造，但镜子不会说谎。

## 核心玩法循环

阅读当晚规则 -> 检查旅客预订、钥匙墙和镜中倒影 -> 允许或拒绝入住 -> 承担判断后果 -> 迎接下一位旅客。

## 视觉 / 美术建议

- 低成本复古旅馆前台界面。
- 暗色纸张、登记薄、监控时间戳、钥匙柜。
- 通过少量静态图片和 UI 异常制造恐怖感，不依赖复杂角色动画。

## itch.io 爆点

- 玩家会主动讨论“第三位旅客到底哪里不对”。
- 规则逐渐令人不安，适合录制短视频和主播口播。
- 允许/拒绝按钮能形成清晰的紧张选择瞬间。

## 开发风险避坑

- MVP 不做自由探索和 3D 场景。
- MVP 不做复杂随机事件生成器。
- 规则变化先用固定数据实现，避免提前设计通用 DSL。
- 恐怖氛围优先依靠文本、音效和轻量视觉反馈。
