# Concepts

These files are visual reference images only. Do not use them as final in-game UI.

Use them to guide:
- lo-fi photo-collage horror mood
- cold green / gray-blue grading
- degraded CCTV / CRT look
- rough painted-over visitor portraits
- motel front desk / hallway environment direction

Recommended workflow:
1. Keep these in `assets/concepts/`.
2. Build actual game UI with HTML/CSS.
3. Put final separated backgrounds in `assets/backgrounds/`.
4. Put final separated visitor portraits in `assets/portraits/`.
5. Put CRT/noise/vignette overlays in `assets/textures/`.
