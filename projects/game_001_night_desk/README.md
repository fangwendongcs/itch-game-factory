# Night Desk / 旅馆前台：请勿让他们入住

`game_001_night_desk` 是 itch.io 小游戏流水线中的第一个真实开发项目。

当前版本处于 **第一版可玩 MVP 阶段**：已经具备 7 个夜晚、15 位来客、四种处置按钮、五项核心数值、即时反馈和多结局。整体方向是原创的暗色前台筛查恐怖小游戏，玩家需要根据登记信息、客人器官异常、影子/倒影破绽和话语矛盾判断是否允许客人进入旅馆。

目标体验时长：15-30 分钟。

## 本地运行

在仓库根目录执行：

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://localhost:8000/projects/game_001_night_desk/
```

## 目录职责

- `data/`: 本游戏专属规则、旅客和结局。
- `assets/`: 本游戏专属音频、图片和 UI 素材。
- `docs/`: 本游戏设计、开发、测试和发布文档。
- 根目录 HTML/CSS/JS: 本游戏可直接运行的 HTML5 源码。

## 当前阶段边界

- 不在 `skills/` 中写本游戏文档。
- 不把本游戏剧情反写到 `templates/`。
- 不在 `releases/` 放开发过程文件。
- 不使用外部依赖。
- 当前已加入极简 favicon、四类低成本音效、concept 参考图和 CSS 生成的场景/人物剪影。
- concept 图只作为风格参考，正式 UI 保持 HTML/CSS/JS 分层，方便后续替换独立人物立绘和背景图。
