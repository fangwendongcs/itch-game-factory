# Rule Judgement Template

可复用的规则判断类 HTML5 小游戏模板。适合制作“阅读规则 -> 检查访客信息 -> 批准或拒绝 -> 查看结果”的短流程游戏。

## 使用方式

1. 复制整个模板目录到 `projects/game_xxx_name/`。
2. 在新项目中修改标题、文案和 `data/` 下的数据。
3. 保留 `templates/` 中的母版为通用版本，不在这里加入某个游戏的剧情或素材。
4. 使用本地静态服务器打开 `index.html`。

## 数据文件

- `data/rules.js`: 规则列表。
- `data/guests.js`: 待判断对象、展示字段和正确决定。
- `data/endings.js`: 根据得分匹配结局。

## 最小机制

- 玩家逐个检查对象信息。
- 玩家选择“批准”或“拒绝”。
- 系统记录判断结果。
- 全部处理完成后显示一个结局。

模板无外部依赖，可直接在现代浏览器中运行。
